# Pharmacy Management System

A console-based Pharmacy Management System developed using Python and MySQL.
This project demonstrates basic CRUD operations and database connectivity.

## Features
- Medicine inventory management
- Staff management
- Customer management
- Billing and sales system

## Technologies Used
- Python
- MySQL
- mysql-connector-python

## How to Run the Project
1. Install Python
2. Install MySQL
3. Run database.sql in MySQL
4. Install dependency:
   pip install mysql-connector-python
5. Update config.py
6. Run:
   python main.py
